○ 주의사항

- [접근성] placeholder 값 사용 시 title 값도 동일하게 제공 예) input type="password" placeholder="비밀번호 재입력" title="비밀번호 재입력"
- [익스 대응] input 등에 readonly가 들어갈 때에는 readonly 클래스도 추가. 예) input type="text" class="readonly" readonly
- [익스 대응] input 등에 disabled가 들어갈 때에는 disabled 클래스도 추가. 예) input type="text" class="disabled" disabled
